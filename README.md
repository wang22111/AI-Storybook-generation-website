## 專題介紹

我們的專題是「AI故事書生成網站」，透過OpenAI模型，提供創作者一個簡單且高效的平台，只需選擇風格，便能根據場景所描述的指定風格圖片，並建立一系列的影片。這個專題的主旨在於幫助那些富有想像力但缺乏繪畫能力的創作者，透過AI技術以圖像形式更生動地呈現他們的故事創作。


## 配置設置

使用以下內容在儲存庫的根目錄中建立本機 .env 檔案：

```bash
OPENAI_API_KEY=YOUR_KEY_HERE---輸入你開通的API金鑰
```

## 檔案須裝的語法
```bash
npm install
```
